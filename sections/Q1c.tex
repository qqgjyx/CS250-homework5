\textbf{Solution:}

\noindent
Extract all the true output cases and take the sum:
$
!A !B !C +
!A  B !C +
 A !B !C +
 A  B  C
$

\noindent
Combining first and third terms
$
(!A + A) !B !C = !B !C,
$
we have:
$
!B !C +
!A  B !C +
 A  B  C
$

\noindent
Combining first and second terms
$
!B !C +
!A  B !C = !C(!A + !B)
$
we have:
$
!C(!A + !B) +
 A  B  C
$
