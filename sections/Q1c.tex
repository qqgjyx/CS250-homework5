\textbf{Solution:}

\noindent
Extract all the true output cases and take the sum:
\[
!A !B !C +
!A  B !C +
 A !B !C +
 A  B  C
\]

\noindent
Combining first and third terms
\(
(!A + A) !B !C = !B !C,
\)
we have:
\[
!B !C +
!A  B !C +
 A  B  C
\]

\noindent
Since $!A !B !C$ already covered in $!B !C$, s.t. $!A  B !C + !A !B !C = !A !C$

\noindent
combining first and second terms
\(
!B !C +
!A  B !C
= !B !C + !A !C
= !C(!A + !B)
\)
we have:
\[
\boxed{
!C(!A + !B) +
 A  B  C
}
\]
